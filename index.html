2025
Pentest SAE
Dans le cadre de ce projet, une maquette réseau préconfigurée m’a été fournie dans le but d’y identifier et d’exploiter un maximum de failles de sécurité. L’attaque débutait par un site web dynamique accessible via le chemin /developper, sur lequel j’ai mené une attaque par force brute avec Hydra. J’ai ensuite exploité une vulnérabilité Log4j pour obtenir un reverse shell. Une fois un accès initial obtenu, j’ai mis en place un mécanisme de persistance via Metasploit, puis réalisé un pivot réseau vers une interface pfSense exposée, accessible depuis le WAN. En progressant, j’ai pu exploiter une faille SMB sur un contrôleur de domaine Active Directory, ce qui m’a permis de récupérer les hachages et de modifier le mot de passe administrateur. L’attaque s’est conclue par un accès à distance en RDP sur une machine du réseau.
2024
Déploiement d'applications communicantes
Projet de fin du premier semestre de ma deuxième année de BUT Réseaux et Télécommunications, ce travail avait pour objectif de développer une application complète de gestion d’emplois du temps basée sur les données fournies par Celcat. Le projet consistait à extraire les données au format XML, les convertir en JSON, puis les stocker dans une base de données MongoDB. Une API développée en Go permettait de requêter ces données de manière sécurisée via SSL/TLS. Enfin, une interface web construite avec Flask offrait une visualisation claire des emplois du temps par salle, incluant les cours en temps réel et des fonctionnalités absentes de l’application Celcat d’origine.
